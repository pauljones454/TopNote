# Design Skills — READ THESE BEFORE WRITING ANY UI CODE

This project uses Taste Skill design directives. Before writing or modifying any frontend component, read all three skill files:

1. `skills/soft-skill/SKILL.md` — Visual language: typography, color, spacing, shadow, motion. The definitive style guide for Top Note.
2. `skills/taste-skill/SKILL.md` — Engineering discipline: component architecture, Tailwind v4 rules, Next.js 16 patterns, strict anti-patterns.
3. `skills/output-skill/SKILL.md` — Output completeness: no stubs, no truncation, no placeholder code.

These are not optional suggestions. They are hard constraints. Every UI component must pass the checks defined in these files.

**Top Note design north star:** Quiet luxury. Feels like the bottles. Editorial, warm, airy. Like a high-end fragrance house, not a social app.
