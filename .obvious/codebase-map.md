# Codebase Map

| Directory | Purpose |
|---|---|
| `app` | Next.js App Router pages and layouts |
| `app/auth` | Supabase auth callback and sign-in flows |
| `app/collection` | User's saved fragrance collection |
| `app/daily` | Daily fragrance feature |
| `app/discover` | Browse and search fragrances by category |
| `app/fragrance` | Individual fragrance detail pages |
| `app/layers` | Fragrance layering — create and explore scent combinations |
| `app/preview` | Studio preview surface |
| `app/profile` | User profile management |
| `app/studio` | iPhone emulator for fragrance studio at /studio |
| `components` | Shared React components |
| `components/layout` | App shell wrapper and sidebar navigation |
| `components/ui` | Reusable UI: BottleImage, FragranceCard, LayerCard |
| `lib` | Supabase client/server helpers, TypeScript types, utilities |
| `public` | Static assets — images and icons |
| `scripts` | Python utility scripts (bottle asset generation) |
| `skills` | Repo-specific design skill files (soft-skill, taste-skill, output-skill) |
| `supabase` | Database migration SQL |
